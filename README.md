# HTML to Figma

将网页内容快速带入 Figma 的 chrome 浏览器插件，适合做参考拆解、局部重设计和界面复用。

## 你可以用它做什么

- 在网页上直接选中任意元素，复制后粘贴到 Figma 继续编辑。
- 拆解复杂页面结构：先选子元素，再逐级选父容器，快速还原布局层级。
- 一次选择多个元素，批量带入 Figma 做重组与 redesign。
- 在查看页面时随时暂停检查模式，临时恢复网页原始交互，再继续选择。

## 交互体验

- 右下角工具条会显示核心快捷键提示。
- 鼠标悬停时当前节点会高亮（蓝色描边 + 浅蓝背景）。
- 在 Flex 布局场景下，会额外提示上层容器（蓝色虚线）。
- 鼠标旁的 badge 会实时提示当前元素类型：`text` / `layout` / `image` / `svg`。

## 快捷键

- `Click`：复制当前元素
- `Shift + Click`：多选 / 取消多选
- `Esc`：选择父级
- `Double Esc`：退出插件
- `Double Cmd`：暂停/恢复 Inspector
- `Enter` 或 `Cmd/Ctrl + C`：复制已选内容

## Figma Capture 说明

- 插件使用 Figma 的 HTML Capture 能力，将选中的网页内容转换为可在 Figma 中粘贴的图层结构。
- 转换结果会受页面结构、字体、样式与站点限制影响，建议粘贴后在 Figma 中继续整理与重设计。
- 不同网站的还原效果可能不同，复杂动效或特殊样式可能出现差异。

## 安装与使用

1. 打开 `chrome://extensions`
2. 开启 `Developer mode`
3. 点击 `Load unpacked`
4. 选择目录 `/Users/carlos/Downloads/HtmltoFigma`
5. 打开任意网页，点击插件图标启动

## 常见提示

- 如果点击插件后没有出现高亮或工具条，先刷新页面再试一次。
- 某些浏览器内置页面（如 `chrome://`）不支持注入插件。
